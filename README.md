# README - 3D Platformer Game

## Project Information
This project is a 3D platformer game created using the Godot Engine.

The game includes:
- Coin collection mechanics
- Timer system
- Double jump
- Main menu with settings
- Fireworks win effect
- Audio and music controls
- Multiple language support

## Controls

- Movement: WASD
- Jump: Space
- Double Jump: Enabled using movement keys (WASD) + Space
- Toggle Mouse Cursor: Assigned in Input Map
